# https://j0.si
a simple link shortener with node.js
### modules required
- express
- express-rate-limit

## Features
- shorten long urls
- links can be migrated from [legacy PHP version](https://github.com/j0-si/j0si-php)
- rate limiting
- domain blacklists (`blacklist.json`)
- custom pathname badwords blocker (`badwords.json`)
