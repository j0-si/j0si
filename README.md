# rewritejs
j0.si rewrited to nodejs from original php
### modules required
- express
- express-rate-limit

## Features
- shorten long urls
- links can be migrated from [PHP version](https://github.com/j0-si/j0si-php)
- rate limiting
- domain blacklists (`blacklist.json`)
- custom pathname badwords blocker (`badwords.json`)
